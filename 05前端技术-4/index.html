jQuery

jQuery是一个JavaScript函数库。jQuery是一个轻量级的”写的少，做的多”的JavaScript库。

jQuery库包含以下功能：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript 特效和动画
  • HTML DOM 遍历和修改
  • AJAX Utilities

除此之外，jQuery还提供了大量的插件。

目前网络上有大量开源的 JS 框架, 但是 jQuery 是目前最流行的 JS 框架，而且提供了大量的扩展。

jQuery 语法

jQuery 语法是通过选取 HTML 元素，并对选取的元素执行某些操作（actions）。

文档就绪事件

文档就绪事件，实际就是文件加载事件。

这是为了防止文档在完全加载（就绪）之前运行 jQuery 代码，即在 DOM 加载完成后才可以对 DOM 进行操作。

如果在文档没有完全加载之前就运行函数，操作可能失败。 所以我们尽可能将所有的操作都在文档加载完毕之后实现。

基础语法 $(selector).action()

选择器 selector

jQuery 选择器基于元素的 id、类、类型、属性、属性值等”查找”（或选择）HTML 元素。 它基于已经 存在的 CSS 选择器，除此之外，它还有一些自定义的选择器。

jQuery 中所有选择器都以美元符号开头：$()。

选择器 语法 说明
标签选择器 $(“标签名”) 基于元素/标签名选取元素
id选择器 $(“#id属性”) 基于id属性选取指定元素
类选择器 $(“.类名”) 通过指定的 class 查找元素
全局选择器 ${“*”} 匹配所有元素
并集选择器 $(“选择器1,选择器2”) 将每一个选择器匹配到的元素合并后一起返回
后代选择器 $(“father son”) 在给定的祖先元素下匹配所有的后代元素
子选择器 $(“father>son”) 在给定的父元素下匹配所有的子元素
相邻选择器 $(“prev+next”) 匹配所有紧接在 prev 元素后的 next 元素
同辈选择器 $(“prev~siblings”) 匹配 prev 元素之后的所有 siblings 元素
属性选择器 $(“选择器[属性]”) 匹配包含给定属性的元素
可见选择器 $(“div:visible”); $(“input:hidden”); 匹配所有的可见或不可见的元素

元素替换

  • replaceWith() -将所有匹配的元素替换成指定的HTML或DOM元素。
