归并排序：从二路到多路

理论

归并排序可用于外部排序

在计算大数据排序的情形下，归并排序有极大优势

外部排序 : 在实现过程中需要借助外存

  • 用2GB内存如何对硬盘中40GB的文件做排序？

    40GB分成20个2GB读入内存做内部排序，得到20个2GB有序文件。

    为每个文件设置一个指针指向当前最小值，内存从20个指针中指向值的最小值读入内存后让指针后移，即做20路归并排序。找20个值中的最小值可以用堆结构。内存读满后重新写入磁盘其他地方得到当前数据最小的2GB的数据。

练习

剑指 Offer 51 逆序对数[困难]

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组，求出这个数组中的逆序对的总数。

0 <= 数组长度 <= 50000

思路

在归并排序的代码中增加计数cnt。对于数组nums[left…right]，mid=(left+right)/2，假设nums[left…mid]和nums[mid+1…right]都已经排好序，并分别用i和j作为归并时指向两个数组的最小值的下标，则在做归并的时候，如果nums[i]>nums[j]，则对于nums[j]，在左半数组中有 (mid-i+1)个数比nums[j]大，而在右半数组中比nums[j]大的都在它右边，故对于nums[j]，计数器仅需加上(mid-i+1)。

LC1305 两棵二叉搜索树中的所有元素 [中等]

给你 root1 和 root2 这两棵二叉搜索树。

* LC327 [区间和的个数 [困难]

给你一个整数数组 nums 以及两个整数 lower 和 upper 。求数组中，值位于范围 [lower, upper] （包含 lower 和 upper）之内的 区间和的个数 。

区间和 S(i, j) 表示在 nums 中，位置从 i 到 j 的元素之和，包含 i 和 j (i ≤ j)。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/count-of-range-sum

提示：

  • 1 <= nums.length <= 105
  • -2^31 <= nums[i] <= 2^31 - 1
  • -105 <= lower <= upper <= 105
  • 题目数据保证答案是一个 32 位 的整数

思路

先获取给定的数组nums的前缀和数组sum，对sum做归并排序。在归并排序的过程中，我们假设左右两半部分都已经排好序，为左右半部分各设置一个下标指针i和j。对于固定的j， 有 lower<=sum[j]-sum[i]<=upper ，即 sum[j]-upper<= sum[i] <=sum[j]-lower 。 因而sum[i]的取值范围仅是sum[j]的函数，在sum[j]确定时sum[i]的取值范围是确定的，而且随着sum[j]的增大而右移。因此，我们设立指针p1和p2来指示sum[i]的取值范围。初始设置p1和p2为左半部分左端点，不断右移找到sum[i]取值范围的端点。对于固定的sum[j]， 所求的区间和个数就是p2-p1。

LC315 计算右侧小于当前元素的个数 [困难]

给你`一个整数数组 nums ，按要求返回一个新数组 counts 。数组 counts 有该性质： counts[i] 的值是 nums[i] 右侧小于 nums[i] 的元素的数量。

提示：

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/count-of-smaller-numbers-after-self

思路

承接上题的思路。假设归并过程中左右半区间已经有序，设置下标指针i和j分别指向左右半数组的首位。固定i，j往右移找到首个不小于nums[i]的j，则 i 位置下右侧小于当前元素的元素个数位j-mid-1。但是由于排序的原因，当前的i不是该数字在原来数组的位置，因此在归并过程中要设置oldToNew和newToOld数组来做某个数字在原数组和归并后数组的双向映射。由于得到oldToNew数组后，newToOld也能决定，所以在归并过程中只需修改oldToNew数组即可。

* LC1508 子数组和排序后的区间和 [中等]

给你一个数组 nums ，它包含 n 个正整数。你需要计算所有非空连续子数组的和，并将它们按升序排序，得到一个新的包含 n * (n + 1) / 2 个数字的数组。

请你返回在新数组中下标为 left 到 right （下标从 1 开始）的所有数字和（包括左右端点）。由于答案可能很大，请你将它对 10^9 + 7 取模后返回。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/range-sum-of-sorted-subarray-sums

提示：

  • 1 <= nums.length <= 10^3
  • nums.length == n
  • 1 <= nums[i] <= 100
  • 1 <= left <= right <= n * (n + 1) / 2

思路

可以考虑暴力求解，这里提供一个k路归并思路解决的方法。

以[1,2,3,4]为例，我们将区间和按以下方式串联成四个链表:

1 -> 3 -> 6 -> 10 ， 2 -> 5 -> 9 ， 3 -> 7 ， 4

我们对这四个链表做四路归并。起初把每个链表的首元结点放到优先队列中，取出堆顶元素后把它的后继结点重新放入堆。我们只用统计第left次取出到第right次取出的结点的数据域的和即是答案。

* 面试题 16.16. 部分排序 [中等]

给定一个整数数组，编写一个函数，找出索引m和n，只要将索引区间[m,n]的元素排好序，整个数组就是有序的。注意：n-m尽量最小，也就是说，找出符合条件的最短序列。函数返回值为[m,n]，若不存在这样的m和n（例如整个数组是有序的），请返回[-1,-1]。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/sub-sort-lcci

提示：

  • 0 <= len(array) <= 1000000

思路

找到所有逆序对的下标中的最大值和最小值。在归并排序的代码中，在比较nums[i]和nums[j]时就能确定这一对数是否是逆序对。
